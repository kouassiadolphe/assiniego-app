# AssinieGo 🇨🇮

**La plateforme qui transforme ton week-end à Assinie en expérience fluide et inoubliable**

Réservez facilement toutes les expériences d'Assinie depuis votre téléphone.

## ✨ Fonctionnalités

- 🏖️ Découverte des activités (Jet-ski, Quad, Bateau VIP...)
- 📱 Réservation instantanée
- 💳 Paiement Wave et Orange Money
- 📲 Installation comme application mobile (PWA)
- 🔄 Mode hors ligne

## 🚀 Démo en ligne

[Voir l'application](https://assiniego-app.vercel.app)

## 📱 Installation sur mobile

1. Ouvrir l'URL dans Chrome/Safari
2. Cliquer sur \"Ajouter à l'écran d'accueil\"

## 🛠️ Technologies

- HTML5 / CSS3 / JavaScript
- PWA (Service Worker)
- Animations CSS
- LocalStorage
